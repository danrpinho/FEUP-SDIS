
Comando para compilar (deve ser corrido dentro da pasta src):
javac -d destDir channels/*.java client/*.java initiators/*.java peer/*.java rmi/*.java utils/*.java 

Comando para iniciar a RMI (deve ser corrido dentro da pasta destDir):
rmiregistry &

Comando para testar um peer (deve ser corrido dentro da pasta src):
java -classpath destDir -Djava.rmi.server.codebase=file:destDir/ peer.Peer  1 1 cenas 224.0.0.0 4445 224.0.0.0 4446 224.0.0.0 4447 


Comando para testar um client (deve ser corrido dentro da pasta src):
java -classpath destDir -Djava.rmi.server.codebase=file:destDir/ client.Client cenas BACKUP test1.txt 1
